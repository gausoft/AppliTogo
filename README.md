## 📖 À propos

**AppliTogo** recense les produits numériques **conçus, développés ou édités au Togo** — applications mobiles (Android / iOS), applications web, plateformes e-commerce et services en ligne.

L'objectif est triple :

- 🔭 **Donner de la visibilité** aux équipes et entreprises tech togolaises.
- 🧭 **Aider les utilisateurs** à découvrir des solutions Made in Togo.
- 🤝 **Valoriser l'écosystème** numérique togolais et ouest-africain.

> 💡 **Vous avez une appli togolaise non listée ?** Ouvrez une [Pull Request](#-contribuer) — la communauté grandit grâce à vous.

## 🤝 Contribuer

Vous avez créé ou repéré une application **Made in Togo** qui n'est pas listée ? Vous êtes au bon endroit !

### En 3 étapes ⚡

```bash
# 1. Forkez ce dépôt sur GitHub, puis :
git clone https://github.com/<votre-user>/AppliTogo.git
cd AppliTogo

# 2. Créez une branche
git checkout -b add/mon-app

# 3. Ajoutez votre app dans la bonne catégorie du README, puis :
git commit -am "feat: ajout de Mon App"
git push origin add/mon-app
# Ouvrez une Pull Request 🚀
```
